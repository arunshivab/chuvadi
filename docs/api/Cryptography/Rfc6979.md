## Remarks

Replaces the per-signature random k with a value derived deterministically from the private key and the message hash. The result is bit-for-bit reproducible (useful for testing and audit), removes the dependency on a strong RNG at sign time, and — crucially — eliminates the catastrophic failure mode where a poor RNG leaks the private key (PlayStation 3 / Sony, 2010).  

 The algorithm is described in RFC 6979 §3.2 and uses HMAC-DRBG with the same hash function the caller is signing under.
