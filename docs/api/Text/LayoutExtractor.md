# LayoutExtractor

**Class** in `Chuvadi.Pdf.Text` (Text)

Extracts text from `TextFragment` objects by reconstructing the visual reading order: group by line (Y position), sort by column (X position).

```csharp
public sealed class LayoutExtractor
```

## Remarks

This strategy is more accurate than `OperatorExtractor` for PDFs where text is not written in reading order in the content stream — for example, multi-column layouts, tables, or PDFs generated by page-layout software that writes content in draw order rather than reading order. Algorithm: 
 
- Group fragments into lines: two fragments are on the same line when their Y difference is less than half the average font size of the line. 
- Sort lines by descending Y (PDF Y increases upward, so highest Y = top). 
- Within each line, sort fragments by ascending X. 
- Insert spaces between fragments based on X gaps.  PDF 32000-1:2008 §9.10 — Extraction of text content.
